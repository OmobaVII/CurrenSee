Key Features

  • Converts Amounts Provided in Whole Numbers of Decimals

    • CurrenSee can convert amounts provided in either whole numbers or decimals and also provides results in two decimal places

  • Last Updated

    • CurrenSee provides users with the last time the exchange rate used for conversion was updated this information allows users to make better decisions

  • Handles Wrong Input

    • CurrenSee can handle input of string and other types other than the required whole number of decimals, it also handles error of inputing negative values

  • Handles Error of Not Selecting a Currency

    • CurrenSee can handle errors of a user not selecting either the base currency or the target currency

  • Handles Error of No Internet Connection

    • CurrenSee can handle a situation where a user does not have internet connection or is using a bad internet connection by providing an error message

About CurrenSee

CurrenSee is a standalone desktop application that converts across multiple currencies, a user just needs to select a base currency, a target currency and the amount he wants to convert. The value of the amount in the target currency is generated.

By focusing on simplicity, usability, and real-time data, my app aims to provide a valuable addition to the existing currency conversion options available to users.

CurrenSee was created as result of the Developer interests in Foreign Exchange as a user who engages in trading of currency pairs on the Forex Market he felt it would be nice to have a simple standalone application on desktop that can easily convert between a wide range of currencies.

CurrenSee was developed as a Portfolio Project for ALX school, the MVP of this project was started and completed within two weeks, by a Sole Developer (Omoba Sanni)
